Sobre esta calculadora

Esta página foi criada como demonstração de uma calculadora de juros compostos em JavaScript puro, com validação e integração com os dados de inflação do Banco Central (IPCA).

A interface prioriza legibilidade e responsividade, funcionando em celulares e desktops. O objetivo é fornecer uma ferramenta educativa para investidores e estudantes.

Você pode ver o código-fonte, adaptar os estilos ou usá-lo como base para projetos maiores. Simples, elegante e sem dependências externas.
