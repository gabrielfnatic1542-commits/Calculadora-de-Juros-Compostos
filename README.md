# 📈 Calculadora de Juros Compostos

Uma ferramenta prática e eficiente para simular o crescimento de investimentos ao longo do tempo. Este projeto foi desenvolvido focado na lógica de **finanças pessoais** e **renda fixa**, permitindo visualizar o poder dos juros compostos no longo prazo.

## 🚀 Funcionalidades

- **Cálculo de Aporte Inicial:** Define o ponto de partida do investimento.
- **Simulação de Aportes Mensais:** Calcula o impacto de depósitos regulares.
- **Flexibilidade de Taxas:** Suporte para juros mensais ou anuais.
- **Detalhamento de Resultados:** Exibe o total investido vs. o total ganho em juros.

## 🧪 Conceito Matemático

O cálculo principal baseia-se na fórmula de juros compostos:

$$M = P \cdot (1 + i)^n$$

Além de considerar a soma de uma série de pagamentos (anuidade) para os aportes mensais:

$$M_{aportes} = PMT \cdot \frac{(1 + i)^n - 1}{i}$$
